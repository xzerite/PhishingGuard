How It Works

Seven-Layer Detection Pipeline

Every URL is passed through a multi-stage analysis pipeline combining rule-based heuristics with a trained machine learning model.

Step 01
📥
Input Parsing
The URL is parsed into its constituent parts — scheme, domain, subdomain, path, query, and fragment — for isolated analysis of each component.
Step 02
🎭
Domain Intelligence
Levenshtein distance is computed against 100+ top brand domains to detect typosquatting and visual impersonation attacks.
Step 03
📐
Heuristic Engine
Rule-based checks flag IP-literal URLs, excessive subdomains, suspicious symbols (@), missing HTTPS, and abnormal URL length.
Step 04
🧠
ML Scoring
A trained model evaluates a feature vector extracted from the URL and outputs a phishing probability score from 0 to 1.
Detection Capabilities

What PhishingGuard Catches

Comprehensive coverage of the most common and sophisticated URL-based attack vectors.

🎭

Typosquatting & Brand Impersonation

Detects domains with small character changes (e.g. paypa1.com, g00gle.com) designed to impersonate trusted brands. Uses Levenshtein edit-distance analysis.

Levenshtein Distance
🖥

IP-Address URLs

Flags URLs using raw IP addresses in place of domain names — a strong phishing indicator used to bypass domain-based blacklists.

Pattern Matching
🔓

Insecure Connections

Warns when a URL uses HTTP instead of HTTPS. Phishing pages frequently skip TLS certificates to avoid cost and detection.

Protocol Check
🎲

High-Entropy Domains

Calculates Shannon entropy of the domain string. Auto-generated malicious domains (e.g. DGA malware) exhibit unusually high entropy values.

Shannon Entropy
🤖

Machine Learning Model

A trained classifier analyses a four-dimensional feature vector — URL length, HTTPS presence, simulated domain age, and form density — to compute a raw phishing probability.

Sigmoid Classifier
🌐

Subdomain Abuse

Excessive subdomain chaining (e.g. secure.login.verify.bank.com) is a common technique to place a trusted-looking term at the front of a malicious domain.
